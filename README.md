# Tripleten web_project_around_auth

**Descripción:**
Este proyecto backend implementa la autenticación de usuarios **(registro, inicio de sesión y verifición de sesión)**

La apliación permite:
- Regustrar usuarios nuevos
- Iniciar sesiín con email y contraseña
- Mantener la sesi+on activa mediante JWT y localStorage
- Proteger rutas privadas

**BASE_URL:**https://se-register-api.en.tripleten-services.com/v1

**Endpoints Utilizados:**

- **POST** http://localhost:3000/signup 
Registro de nuevos usuarios

- **POST** http://localhost:3000/signin
Autorización de usuarios y obtención del token JWT

- **GET/users/me** http://localhost:3000/users/me Verificación del token y obtenciín del email y del usuario autenticado



**Tecnologías utilizadas:**
**React:** Construcción de la interfaz y manejo del estado

**React Router DOM:** Navegación y protección de rutas

**JavaScript (ES6+)** 

**CSS:** Estilos basados en Figma

**JWT (JSON Web Token):** Autenticación basada en tokens

**Vite:** Entorno de desarrollo
